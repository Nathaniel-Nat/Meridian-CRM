# CRM Portfólio Pessoal

Sistema de gestão de relacionamento com clientes (CRM) simplificado, desenvolvido para portfólio.

## 🚀 Tecnologias
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- LocalStorage para persistência de dados

## ✨ Funcionalidades
- **Dashboard**: Visão geral de métricas (Total de contatos, negociações abertas, taxa de conversão).
- **Gestão de Contatos**: CRUD completo de leads e clientes com busca e filtragem.
- **Pipeline Kanban**: Gestão de negociações com Drag-and-Drop nativo entre etapas.
- **Histórico de Atividades**: Registro de interações (ligações, e-mails, reuniões) vinculadas a cada contato.

## 🛠️ Como executar
1. Clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Os dados são salvos automaticamente no `LocalStorage` do seu navegador.

## 📦 Deploy
Este projeto foi configurado para deploy via **GitHub Pages**. Para publicar:
1. Faça o push do código para um repositório no GitHub.
2. Vá em `Settings` > `Pages`.
3. Selecione a branch `main` e salve.
